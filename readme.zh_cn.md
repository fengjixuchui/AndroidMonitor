## 特性
- 遵循谷歌材料设计语言
- 帮助开发者了解Android系统
- 查看到设备的隐藏信息

## 开发
- 无架构开发
- 抓住重点，其他都忽略
- 代码足够少，功能却不少
- 用邮箱来实现意见反馈功能
- 尽量不使用非官方库，个人项目，不考虑多人协作，所以不使用常见通用的第三方库。
- 朴实的代码

## 迭代
- V1.0
```
管理设备应用
查看设备硬件信息
```
